# 概要
[このページ](https://kohimoto.com/labo/web/usability/18790/)の下部ではスクロールしていったときにとても自然にダークテーマからホワイトテーマに切り替わります．このようなインタラクションを私のプロジェクトでも取り入れたいと考えています．私のプロジェクトは，**基本はダークテーマに強制**しており，Chakra UI V3を利用したカスタムコンポーネント`src/components/layouts/SectionContainer.tsx`に与えるプロップスによってダークテーマかライトテーマか指定する実装になっています．現状，aboutセクションとstoryセクションでのみこのような疑似的なライトテーマを使用したいと考えています．topセクションからaboutセクションに移動するとき，ダークからライトに自然に切り替わり，aboutセクションからproductセクションに移動するときに自然にライトテーマからダークテーマに遷移するインタラクションを取り入れたいです．また同様に，skillsセクションからstoryセクションに移動するときもダークテーマからライトテーマに自然に遷移するようにしたです．

現状の実装だと，Topセクションから少しだけ下に行くとダークテーマと，次のセクションのライトテーマの境目がくっきりしすぎているんですよね．(背景にグラデーションをつけろというわけではないです．)参考にしたサイトみたいに，一定のところまでスクロールしてきたタイミングで背景がしゅっとダークテーマからライトテーマに切り替わるようにしたいです．
