# はじめに
あけましておめでとうございます．
これまでかかわってくださった方々2026年もよろしくお願いします．


年末年始は，来年度から新生活を始める家族と実家向けにラップトップの新調任務があったため，秋葉原の初売りで大好きThinkPad(八世代i5 X1を2万円)を調達してクイックレスキュー環境を整備していました．


他にも親戚周りへの挨拶周りや，開講期間中から積んでいた本の消化やRuby再入門(現在も継続中)などいろいろあったように思います．


世間が仕事初めの中，もう少しだけのんびりできる正月期間があったため今日は地元で走りなれた道をのんびり60kmほどサイクリングに行ってきました．


この記録では地元のお気に入りスポットと最後には2026年の抱負を残しておきます．今後弱音を吐いてたらここに残した抱負を送り付けてやってください...

# 自転車はイイぞ
今回の目的地はジャパンカップサイクルロードレースのコースにもなっている宇都宮森林公園 古賀志山．10月に開催されたレースで整備された林道も冬の寒さで枯れた草葉に覆われていました．また年始に関東全域で振っていた雪もところどころ残っていて前回ゴールデンウイークに訪れた夏の景色とはガラッと変わっていましたね．


久方ぶりに田舎の道を走って思うのは圧倒的ストレスのなさ．千葉で20kmすすもうものなら信号だらけで30分～1時間程度かかる中，森林公園を目指す農道では信号どころかコンビニもないのでノンストップで走り抜けれます(もちろん風よけなんてものはないので気合も必要です)

地元のコミュニティでもよく一緒に走るエリアの入口．日光東照宮方面への車通りの少ないアクセスになるほか，70kmの周回路がほぼ信号なしで走れるため昔はよく来ていた．


今日は思いのほか寒さと久しぶりのミドルライドで脚に応えたので今日はここまでで折り返し．

# おわりに
走行距離としては60km,600m登坂,4時間弱のライドとなりました．久々の地元で懐かしい気持ちになりながらもいい気分転換になりました．


やはり自転車でのんびり走っていると考え事や気持ちの整理が出来て心身共に健康的な趣味だなと感じています．大学の開講期間も終了し，年末年始は自身のキャリアや人生設計についていろいろ考えていました．趣味でやってたモノづくりを仕事にしていけるのか，いつまで探究心を持って最前線で走り続けられるのだろうかなど色々考えていました．自転車というもう一つの趣味をやりながらも今日改めて思ったのは，自分にとって仕事は[Rice Work](https://diamond.jp/articles/-/189534)ではなくLife Work(人生を豊かにする自己実現の仕事)として生涯をかけて熱中したいものでありたいと思っています．


これまでやってきた，先代のエンジニアの方々が築き上げてきた技術をもっと身近で味わいたい．そして自分自身も人生の中で技術者に使われる技術を後世に残したいという思いがあります．その過程として従事するライフワークにおいても技術を学び続けられる環境に居続けること．そして何よりも，これまで開発をしてきた原動力でもある，「技術で誰かの不便や課題を解決したり，面白い，楽しいを届けられるようなプロダクトづくり」を自身の職していく覚悟が出来ました．


## 2026年の豊富
今年の抱負は3つです．
- 多くの人とかかわること．
- 毎日小さくても昨日の自分よりも成長すること(技術的にも人間的にも日々精進)．
- 無病息災．元気でいること．

これらの抱負の理由と具体的に何を目標とするのか．一つ目の抱負に関しては　研究テーマにしても個人開発にしても，ニーズや課題を知るためにはこれまで身近だったコミュニティ以外の人たちの考えや価値観を知る必要がある．学生という立場で比較的自由にいろいろなことに挑戦できる(金銭的な制約こそあるものの失敗した時のリスクがまだ軽微)時期だと思ってます．残り1年間の学生生活の中で自身の研究テーマの探求や興味を抱いた分野の技術だけでなく，これまで関わってこなかった分野に関する情報についても知る努力をしたいと思っています(これまで興味を持たなかった分野は触れずに知識が偏りがちだったので)．


二つ目の抱負については技術者としての精進です．Web領域に進むうえで，ここまでWeb開発の基盤を支えてきた基礎技術．また，かなりの速さで変化し続けるその時々に合わせたトレンド技術のキャッチアップなど，まだまだ知らないことと知りたいことが山積みです．今年はRialsなど長きにわたってWeb開発のスタンダードとなってきた技術を体系的に学びつつ，ブログや個人開発といった形で学びをアウトプットして形にしていきたいと思います．

また，一つ目の抱負ともつながりますが，今年は学外の技術系イベントに積極的に参加して同じ技術領域で活躍されている方々との交流を通じてこれまで一人では見ることのできなかった価値観や知見を得るなどしたいと思ってます(自分から人前に立つのが苦手な性格故一歩目が重いですが頑張りたいです)．


最後の抱負は言わずもがな．自身の身体あっての人生，食生活や睡眠サイクル，適度な運動といった習慣を見直し健康な一年を送るために尽力します．2026年1月時点でも，AIエージェントがやってくれることが増え，AIが書いたコードの理解に集中力と体力が求められていることを実感してます．AIと伴走しながらその時々にあった開発についていくためにも，楽しみながら体力維持を継続していきます．



これらの抱負を掲げて今年も一年頑張ります．みんなで悔いのない最後の学生生活を完走するぞ...!
